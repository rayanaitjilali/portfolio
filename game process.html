Concept & Vision

Al's Quest: The Charij Chronicles was born from a desire to create a game that celebrates Moroccan culture while delivering an engaging gaming experience. The story follows Al Amrani on his quest to save Zaribt village from a devastating drought.

  • → Moroccan-inspired world and characters
  • → Four unique worlds with distinct bosses
  • → Simple yet engaging gameplay mechanics

Design & Development

The development process focused on creating a lightweight yet visually appealing game using web technologies.

  • → Initial sketches and character designs
  • → Figma for UI/UX and asset creation
  • → JavaScript, HTML, and CSS implementation
  • → Mobile-first approach for wider accessibility

The Four Worlds

Each world in Al's Quest presents unique challenges and encounters:

  • → Gazoo's Domain - The Desert Trials
  • → Bouslham's Lair - The Mountain Challenge
  • → Habira's Realm - The Forest Mystery
  • → Bouzbal's Territory - The Final Test

Technical Implementation

The game was built with modern web technologies to ensure smooth performance and easy accessibility:

  • → Responsive design for all screen sizes
  • → Optimized animations and transitions
  • → Efficient collision detection system
  • → Cross-platform compatibility
